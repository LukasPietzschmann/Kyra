# Contributing to Slanguage

## I don' want to contribute. I have a question
The issue-tag **question** is here for you.

## I have a new idea
Just go ahead and create an issue tagged with **idea**, where you describe your idea.

What you should include in the issue:
- Detailed explanation of what to add
- Why do you want this idea to become a feature
- Some examples of how the syntax should look like

## I want to implement my idea
Read [I have a new idea](https://github.com/LukasPietzschmann/Slanguage/blob/master/CONTRIBUTING.md#i-have-a-new-idea) and do what is described there.
Then, if we agreed on what exactly the feature should look like, go ahead and file a PR.

## I found a bug
Just create an issue tagged with **bug**.
